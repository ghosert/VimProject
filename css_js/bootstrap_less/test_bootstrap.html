Enabling responsive features

Add two sentences below to <head> tag, then the page will be responsive based on different devices, resolutions.
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<link href="bootstrap/bootstrap/css/bootstrap-responsive.min.css" rel="stylesheet">

media queries in a single file to help make your projects more appropriate on different devices, resolutions.

Usage

Enable tabbled tabs via Javascript

                $('#myTab a').click(function (e) {
                    e.preventDefault();

Markup

If you want to activate a tab or pill navigation without writing any JavaScript, specify data-toggle="tab" / "pill" on an element.

                <ul class="nav nav-tabs">
                    <li><a href="#home" data-toggle="tab">Home</a></li>
                    ...
                </ul>
href="#home" here is the div id in the div.tab-content #home which contains the content to be shown when clicking on home tab
see the source code sample above.

Methods

$().tab

Activates a tab element and content container. Tab should have either a data-target or an href targeting a container node in the DOM.

                $('#myTab a:last').tab('show'); // show tab manually by JavaScript

Events

show, shown
                $('a[data-toggle="tab"]').on('shown', function (e) {
                    e.target // activated tab
                    e.relatedTarget // previous tab
                })

Buttons

Example uses

Stateful

Add data-loading-text="Loading..." to use a loading state on a button.

Single toggle

Add data-toggle="button" to activate toggling on a single button.

Checkbox

Add data-toggle="buttons-checkbox" for checkbox style toggling on btn-group.

Radio

Add data-toggle="buttons-radio" for radio style toggling on btn-group.

Usage

$().button()

Methods
$().button('toggle')

Same as data-toggle="button" above

$().button('loading')

Sets button state to loading - disables button and swaps text to loading text. See samples above.

$().button('reset')

Resets button state - swaps text to original text.

$().button('complete')

Resets button state - swaps text to any data defined text state.

Usage

Via JavaScript
                $().carousel()

You can check out http://twitter.github.com/bootstrap/assets/js/application.js

Options

interval pause: See docs

Methods
$().carousel(options)

$().carousel({interval: 2000})

$().carousel('cycle')

Cycles through the carousel items from left to right.

$().carousel('pause')

Stops the carousel from cycling through items.

$().carousel(number)

Cycles the carousel to a particular frame (0 based, similar to an array).

$().carousel('prev')

Cycles to the previous item.

$().carousel('next')

Cycles to the next item.

Events

slide, slid
