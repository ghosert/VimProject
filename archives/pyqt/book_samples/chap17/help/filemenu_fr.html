File|New Create a new plain image.

File|Open Open an existing image.

File|Save Save the current image.

File|Save As Save the current image under a new name, and make the newly named image the current image.

File|Quit Terminate the program. You will be given the chance to save any unsaved changes.

If there are numbered files, these are recently opened files. Choosing one of them will cause it to be opened and to become the current image.
